# Apprendre le Creole

A simple learning app for Haitian Creole, available on web and ready to be packaged for Android/iOS.

## Features
- Language switcher (English / Kreyol Ayisyen)
- Vocabulary categories
- Pronunciation with text-to-speech
- Quiz, matching, and speaking practice
- Contact and support section

## Run (Web)
Open `index.html` in your browser.

## Project Files
- `index.html`
- `style.css`
- `script.js`

## Mobile Build (Capacitor)
1. Install dependencies:
```bash
npm install
```
2. Add platforms:
```bash
npx cap add android
npx cap add ios
```
3. Sync and open:
```bash
npx cap sync
npx cap open android
npx cap open ios
```
